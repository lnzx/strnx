# strnx

主要用来监控钱包，查看每日收益，节点运行状态, 适合很多钱包的用户。

默认端口8080,无法更改。

## env (环境变量)

`DATABASE_URL` 

e.g.

`postgresql://USER:PASSWORD@127.0.0.1:PORT/defaultdb?sslmode=[disable|verify-full]`

`KEY` 密码,默认123456

可选参数，发送短信相关配置:

`SMS_API_KEY` 发送短信平台的api key,另一个项目。

`MOBILE` 接受短信手机号，如: 13988888888
